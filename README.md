# DWES-JMO
Desarrollo Web Entorno Servidor

## Sobre el curso
Repositorio para la asignatura de Desarrollo Web en Entorno Servidor (2º DAW). Aquí encontrarás ejemplos y prácticas progresivas para aprender a crear servicios backend modernos con JavaScript/Node.js.

## Objetivos
- Entender la base del entorno servidor con JavaScript y módulos nativos.
- Aprender a estructurar proyectos con Express, middlewares y rutas.
- Trabajar con pruebas (unitarias y funcionales) para asegurar calidad.
- Automatizar tareas con npm scripts y aprovechar el ecosistema de paquetes.

## Estructura de carpetas
- `02-` a `06-`: Fundamentos de Node.js, módulos y patrones iniciales.
- `07-` a `11-`: Depuración, tests y buenas prácticas.
- `12-` a `17-`: Servidores HTTP, Express, rutas y middlewares.
- `18-` en adelante: APIs REST, tests funcionales y mejora de arquitectura.

## Requisitos básicos
- Node.js LTS instalado.
- Editor con soporte para JavaScript/TypeScript y ESLint (ej. VS Code).
- npm para instalar dependencias y ejecutar scripts.

## Cómo usar este repositorio
- Lee cada carpeta en orden y revisa el código comentado.
- Ejecuta `npm install` en los ejercicios que lo requieran y luego `npm test` o `npm run dev` según la práctica.
- Ajusta ejemplos y experimenta para reforzar conceptos.
